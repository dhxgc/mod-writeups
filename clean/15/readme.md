### 3 вопрос:
```text
Протокол SNMP (Simple Network Management Protocol) — стандартный интернет-протокол для управления устройствами в IP-сетях.

Компоненты:

Управляемое устройство — устройство, которым управляет SNMP (например, маршрутизаторы, принтеры).

Агент — программа на управляемом устройстве, передающая данные.

Менеджер — административная система, управляющая устройствами через SNMP.

Набор услуг:

- GetRequest — получение данных с устройства.

- SetRequest — изменение данных устройства.

- GetNextRequest — получение следующей переменной.

- GetBulkRequest — запрос нескольких переменных.

- Trap — уведомление об ошибке или событии.

- InformRequest — уведомление с подтверждением получения.

Базы MIB — иерархическая структура данных устройств, используемая для управления через SNMP.

Характеристики:

- Работа на уровне приложений (7-й уровень OSI).

- Использует UDP/TCP (порты 161, 162).

- Асинхронные уведомления (Trap).

- Иерархия данных (MIB).

- Версии: SNMPv1, v2, v3 (с улучшенной безопасностью).
```
